Integración Continua con GitHub Actions
Proyecto de demostración de CI/CD utilizando GitHub Actions para ejecutar un programa y enviar notificaciones automáticas a ntfy.sh.
Descripción
Este proyecto implementa un flujo de integración continua que:

Ejecuta un programa JavaScript al hacer push a la rama main
Envía notificaciones automáticas a ntfy.sh/devops-itla
Notifica tanto en caso de éxito como de fallo
